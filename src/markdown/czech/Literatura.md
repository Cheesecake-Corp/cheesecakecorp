# Literatura (písemnictví)

- umělecká, odborná, administrativní
- umělecká
  - ústně lidová slovesnost
  - umělá dála - napsaná autorem

## Vědy o literatuře

1) literární historie
2) literární teorie
3) literární kritika

## Literární žánry

1) lyrika - popis pocitů
   - milostná
   - přírodní
   - vlastenecká
2) epika - zaměření na děj
3) drama - vyprávění pomocí monologů a dialogů

- mohou být psány veršem i prózou

## Termíny

- poesie - psáno veršem
- próza - souvislý text
- elipsa - neúplná věta
- legenda - středověká skladba obsahující vypravování o životě světců
