# Neuron

- NS člověka obsahuje cca 20-25 miliard neuronů
- **funkce:** tvorba a přenos nervových signálů

## Stavba

- **dendrit**
  - krátký výběžek, spolu s tělem - vstupní část neuronu = místo příjmu a zpracování signálů z jiných neuronů
- **tělo neuronu (soma)**
- **iniciální segment**
  - vznik akčních potenciálů (AP)
- **axon (neurit)**
  - dlouhý výběžek
  - vede AP od těla neuronu
- **myelinová pochva**
  - obaluje axon
  - tvořena Schwannovými buňkami
  - přerušovaná Ranvierovými zářezy -> urychlení přenosu AP
- **nervová zakončení**
  - tvoří výstupní část axonu
  - tzv. **SYNAPSE** = místo přenosu signálu mezi neruony nebo mezi neuronem a cílovou buňkou
  - "knoflíkovitý" tvar
